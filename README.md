# 四级高频词自测

基于 SM-2 间隔重复算法的四级高频词智能自测工具。

**在线使用**: https://qqqqqqiu0804.github.io/cet4-words-quiz/

## 功能

### 刷题
- 9大分类：时政、科技、社会文化、校园、医疗、经济、环保、写作、科研
- 中译英 / 英译中双模式
- 顺序 / 随机刷题
- 难度筛选（★ / ★★ / ★★★）
- 答错照打正确答案 + 音效反馈

### 智能复习（SM-2）
- SM-2 间隔重复算法，根据回忆速度自动安排复习时间
- 智能复习模式，优先抽取到期需复习的单词
- 单词掌握程度标签（新词 → 初学 → 熟悉 → 掌握 → 熟练 → 精通）
- 复习进度面板，查看整体掌握情况

### 学习管理
- 错题本 + 收藏夹（支持导入导出）
- 今日目标 + 每日统计
- 🔥 连续学习天数
- 答错自动展示例句 + 发音

### 个性化
- 日间 / 夜间 / 护眼三种主题
- 字体大小调节（小 / 中 / 大）
- 键盘快捷键

## 快捷键

| 按键 | 功能 |
|------|------|
| Enter | 提交答案 |
| Esc | 清空输入 |
| ← → | 上/下一题 |
| Space | 随机抽题 |
| S | 发音 |
| F | 收藏 |
| R | 复习进度 |

## 技术

- 纯前端，单个 HTML 文件，无需后端
- 数据存储在 localStorage，支持导入导出
- PWA 支持，可添加到手机主屏幕
