## 一、项目概述

本项目是一个专为大学英语四级考试设计的高频词智能自测工具。不同于传统的单词 App，它将 SM-2 间隔重复算法融入刷题流程，实现"学-练-复习"的完整闭环，帮助用户高效记忆四级高频词汇。

**技术栈**：纯前端（HTML + CSS + JavaScript），单文件部署，零后端依赖。

**数据存储**：localStorage 本地持久化，支持 JSON 导入导出。

**部署方式**：GitHub Pages 静态托管，PWA 支持添加到手机主屏幕。

### 2. SM-2 间隔重复算法

SM-2 是 SuperMemo 经典的记忆算法，核心原理：

- **答对**：根据回忆速度评分（<2秒=5分，<5秒=4分，否则=3分），连续答对则复习间隔指数增长（1天→6天→按易度因子倍增）
- **答错**：重置间隔为1天，重新开始记忆周期
- **易度因子（EF）**：动态调整，难度大的单词复习更频繁

每个单词独立追踪：重复次数、复习间隔、易度因子、下次复习时间。

### 3. 智能复习模式

开启"🧠 智能复习"后，系统优先抽取到期需复习的单词，而非随机出题。配合掌握程度标签（新词 → 初学 → 熟悉 → 掌握 → 熟练 → 精通），一目了然地看到每个单词的记忆状态。

### 4. 学习数据可视化

- **GitHub 风格热力图**：365 天学习记录，绿色深浅表示每日学习量
- **连续学习天数（Streak）**：激励每日坚持，中断归零
- **每日统计**：已答/正确/正确率实时更新
- **复习进度面板**：查看整体掌握情况和待复习单词列表

### 5. 学习管理

- **错题本**：自动收集答错单词，支持重做和清空
- **收藏夹**：手动收藏重点单词，随时复习
- **导入导出**：错题本和收藏均支持 JSON 格式导入导出，数据不丢失
- **答错自动例句+发音**：答错后自动展示例句并朗读单词，多感官辅助记忆

### 6. 个性化体验

## 三、技术亮点

### 架构设计
```
单文件 PWA 架构
├── index.html      — 所有逻辑（HTML + CSS + JS）内联
├── manifest.json   — PWA 清单，支持安装到主屏幕
└── sw.js           — Service Worker，离线缓存
```

### 核心算法
```javascript
// SM-2 间隔重复算法
function sm2(wordKey, quality) {
  if (quality >= 3) {
    // 良好回忆：间隔增长
    interval = repetition === 0 ? 1
             : repetition === 1 ? 6
             : Math.round(interval * efactor);
    repetition++;
  } else {
    // 不良回忆：重置
    repetition = 0;
    interval = 1;
  }
  // 更新易度因子
  efactor += 0.1 - (5 - quality) * (0.08 + (5 - quality) * 0.02);
  if (efactor < 1.3) efactor = 1.3;
}
```

### 数据模型
- **词库**：`{ en, cn, cat, difficulty, example, note }` — 覆盖9大分类的四级高频词
- **SM-2数据**：以 `en+'|'+cn` 为键，存储 `{ repetition, interval, efactor, nextReview, lastReview }`
- **每日记录**：`dailyHistory` 存储每天答题数，驱动热力图
